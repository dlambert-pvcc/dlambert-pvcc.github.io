List four disadvantages of using assembly language:
1. Pseudo-manual movement of data between locations and registers.
2. Tasks are broken down into incredibly tiny components.
3. It's machine specific.
4. It's way harder than reading something like English or Spanish.

List four expectations programmers have for writing programs in a high-level language ("third generation" language):
1. Manual memory allocation is unnecessary.
2. Their building blocks are larger.
3. They're not machine specific.
4. They're closer to a natural language, with recognizable math.

Briefly describe or define:
1. object code
"[B]inary machine language instructions".
2. source code
"[H]igh-level language instructions".
3. procedural language
"[...] consists of sequences of statements that manipulate data items." Step by step instructions.
4. Integrated Development Environment (IDE)
A combined text editor, compiler, linker, and debugger, frequently with a lot more extra features. What I'm writing this in.
5. software development life cycle
"[T]he overall sequence of steps needed to copmlete a large-scaled software project."
6. waterfall model of software development
One phase before the next, before the next, before the next. Each phase must be completed before moving on.
7. agile model of software development
Flexibility in design and editing. Readiness to go back and make changes to previous phases.

Describe each of these components of the software development life cycle:
1. feasibility study
Project proposal evaluation; cost/benefit analysis.
2. problem specification
"[D]eveloping a clear, concise, and unambiguous statement of the exact problem the software is to solve."
3. program design
Figuring out how to do the above before getting started on it.
4. algorithm selection
Picking the best algorithms for the job (and each sub-job).
5. coding
The fun stuff. Converting designs into computer code.
6. debugging
The miserable stuff. Fixing errors in the code.
7. testing
Making sure the program or module does what it's actually supposed to. Should be done through coding and debugging.
8. verification
Checking that inputs and outputs behave as intended as long as the inputs are correct.
9. documentation
This. But in unambiguous words.

Chapter 10: Tower of Babel: Programming Languages
Questions & Answers:
Q: Why does the textbook refer to "the Tower of Babel" regarding computer languages?
A: References a Bible story. There's too many programming languages to learn them all anymore.
Q: Why isn't there just one computer proramming language?
A: Different languages have different kinds of tasks they're suited for. Not all are equally suited for the same problem.
Q: Why didn't FORTRAN use the less than ( < ) symbol when comparing two numbers?
A: "[T]he keypunches that were used to create the punched cards on which early Fortran programs were submitted to the computer had no such symbols."
Q: What was COBOL designed for?
A: Business needs, for business people. Inventory and payroll.
Q: What was the language, C, designed for?
A: Writing UNIX.
Q: What is the difference between C and C++?
A: Everything that can be done in C can be done in C++, but not everything that can be done in C++ can be done in C. C++ has additional and newer features, like object-oriented programming.
Q: Who or what is the language, Ada, named for and why?
A: Ada Lovelace, the world's first programmer.
Q: What is a Java "applet"?
A: Programs specifically for webpages.
Q: How much does Python cost to download, install, and use?
A: It's completely free. I've used it in a couple classes, it's nice.
Q: What is Microsoft.NET?
A: A huge framework of tools for C# and several other languages. "It was designed so that traditional text-based applications, GUI applicattions, and web-based programs can all be built with equal ease."
Q: What is SQL and what is it used for?
A: Structured Query Language, a language designed specifically for databases. I'm on the side that pronounces it as "sequel."
Q: What is R and what is it used for?
A: Another specialized language, this time for stats and graphics
Q: What is Go (or golang) and what is it used for?
A: Google's language. Meant to offer ease of programming without sacrificing efficiency and safety of execution.
Q: What is Swift and what is it used for?
Apple's language for iOS development. Now works on Linux as well.

Briefly describe or define:
1. functional programming
"A functional programming language views every task in terms of (surprise!) functions." Offers built-in math functions for programmers.
2. logic programming
"In logic programming, various facts are asserted to be true, and on the basis of these facts, a logic program can infer or deduce other facts."
3. parallel programming
"Parallel processing is really a catchall term for a variety of approaches to computing architectures and algorithm design." Multiple processors handle different subtasks simultaneously.
