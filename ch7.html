Briefly describe or define:
1. computer network
A "set of independent computer systems interconnected by telecommunication links for the purpose of sharing information and resources."
2. analog device (define and give an example)
Uses a "continuous physical quantity" (voltage or otherwise) to represent information, like a Timex powered by a battery.
3. digital device (define and give an example)
Uses binary to represent information, like a smartwatch or a Timex with a screen.
4. modem
"[M]odulates, or alters, a standard analog signal called a carrier wave so that it encodes binary information."
5. bandwidth
The "rate at which information [can] be sent and received".
6. ethernet
Originally coaxial cable, now also fiber optic or twisted-pair copper wire. Nowadays, 1 Gbps transmission speed or greater.
7. WLAN
Wireless local area network. For homes, offices, or small shops.
8. IEEE 802.11 wireless network standard
Wi-Fi. Connects computers to the Internet when in range of a router, usually 150-300 feet. Usually uses 2.4 GHz radio band.
9. MAN
Metropolitan area network. Few city blocks' worth of range, or a neighborhood network.
10. Internet of Things
Somebody's refrigerator is now connecting to the Internet just like you.
11. LAN
Local area network. Owner of the devices using the LAN owns the means of communication as well, like the cables and network bridges.
12. repeater
"[A] device that simply amplifies and forwards a signal." Can connect together multiple LANs.
13. bridge
Smarter than repeaters. Checks for whether or not it needs to pass data between LANs before doing so.
14. WAN
Wide area network. Dedicated point-to-point lines that connect two far-apart devices directly. Data hops along different nodes to reach destination.
15. packet and packet switching
Packet: "an information block with a fixed maximum size that is transmitted through the network as a single unit." Packet switching: the method of transmitting packets across different nodes to their same destination.
16. ISP
Internet service provider. Businesses (frequently local monopolies) that connects private networks to the Internet.
17. firewall
"[H]ardware and software placed at the entry points to a company's [or individual's] private network to protect it from unauthorized access."
18. protocol
Networked machines' equivalent to a social contract. "[A] mutually agreed-upon set of rules, conventions, and agreements for the efficient and orderly exchange of information."
19. cloud computing
Alternative to the client/server model with a name that truly embodies the opaqueness of what companies do with the private information we provide them. Ahem: "[W]ith cloud computing the servers no longer need to be local to the client population, and they no longer must be provided by your own organization." Prepackaged remote server service.
20. net neutrality
"[A] public information network such as an Internet sercice provider (ISP) should treat all users, all platforms, and all content equally--essentially it should be operated like the water, electric, and gas companies, as publicly regulated utilities."

Briefly describe the five layers of TCP/IP (the Internet protocol hierarchy):
Layer 1. Physical layer
"[G]overn(s) the exchange of binary digits (bits) across a physical communication channel". Cable or wireless protocols, or the "bit pipe."
Layer 2. Data link layer
Figuring out the start and end of messages, dealing with errors in transmission, transmission collision handling.
Layer 3. Network layer
The IP in TCP/IP, for Internet Protocol. Ensures each node in a network runs the same protocols and delivers between nodes in the same network.
Layer 4. Transport layer
The TCP in TCP/IP, basically the node postal service. In charge of assigning ports within the same IP address for more specific data transmission to programs.
Layer 5. Application layer
End-user rules. Several separate protocols for governing how a user accesses webpages, sends and receives emails, accesses remote files or terminals, and translates IP addresses.
