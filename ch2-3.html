2. Problem 1:
A student has three grades; the final grade is the average of the three. List the algorithmic steps to find the final grade.
  1. Get values for grade 1, grade 2, grade 3
  2. Set value of grade total to (grade 1 + grade 2 + grade 3)
  3. Set value of grade average to (grade total ÷ 3)
  4. Print the value of grade average
  5. Stop
3. Problem 2:
A car owner recorded the car's previous car mileage, current mileage, and the number of gallons of gas the owner filled their tank with. List the algorithmic steps to find the car's mpg (miles per gallon).
  1. Get values for gallons used, starting mileage, ending mileage
  2. Set value of distance driven to (ending mileage - starting mileage)
  3. Set value of average miles per gallon to (distance driven ÷ gallons used)
  4. Print the value of average miles per gallon
  5. Stop

Chapter 2: Briefly define:
1. natural language
"[T]he language we speak and write in our everyday lives."
2. pseudocode
Natural language written and ordered to resemble a programming language.
3. sequential operation (straight-line algorithm)
Do x, then do y, then do z.
("[I]t executes its instructions in a straight line from top to bottom and then stops.")
4. input
Data from the outside world provided to the computer.
5. output
Data sent from the computer to the outside world.
6. conditional statement
Do x if y.
("[A]sk a yes/no question and select the next operation to perform on the basis of the answer to that question.")
7. boolean condition
True or False.
(If y is True, do x.)
8. algorithmic discovery process
"Finding a solution to a given problem".

Chapter 3: Briefly describe or define:
1. why algorithms should be correct
If they're incorrect, they "can lead to mistakes that are enormously expensive or even fatal."
2. why algorithms should be efficient
"[T]ime and space are not unlimited resources." Using fewer computer resources allows the algorithm to be used in more situations or on weaker computers.
3. program maintenance
Fixing errors as they're discovered; extending program to meet new requirements.
4. importance of algorithm: ease of understanding
"To make program maintenance as easy as possible".
5. algorithmic elegance
"[T]the algorithmic equivalent of style." Sometimes at odds with ease of understanding.
6. sequential search algorithm
Searching one by one through a list for a match in set data (a number, word, phrase, etc.).
7. order of magnitude
"Anything that varies as a constant times n". An algorithm with the lowest order of magnitude requires the least time to execute.
8. selection sort
"The selection sort 'grows' a sorted subsection of the list from the back to the front" so as to sort of a list of items into order.
9. flop (floating point operation)
Arithmetic operations on real numbers.
10. teraflop
A trillion floating-point operations per second.
11. petaflop
A quadrillion floating-point operations per second.
12. parallel processor
Multiple processors working on different problems or smaller divisions of a larger problem simultaenously.
13. data cleanup problem
The issue of how best to remove unwanted data from a list.
14. binary search algorithm
Search middle of list for x; if selection is x, search is over. If selection is higher/lower than x, remove lower/upper half of list. Search middle of new list for x; if selection is x, search is over. Et cetera.
15. pattern matching
Find all instances of pattern x within text y.

1. Give a brief overview of Karl Friedrich Gauss's mathematics abilities and scientific achievements.
As explained by Encyclopedia Britannica, he could do elaborate calculations in his head. His achievements include publishing the first systematic textbook on algebraic number theory and his rediscovery of the asteroid Ceres, a feat in calculating the asteroid's trajectory after it was first lost behind the sun.

The highest and lowest numbers, second highest and second lowest numbers, etc., can all be summed to 101. That makes 50 pairs of 101. Therefore, the solution equals (100 ÷ 2) x 101, or, as long as the highest value is an even number, (highest number ÷ 2) x (highest number + lowest number).
3. What are gaussian optics?
According to RP Photonics Encyclopedia, "a framework for describing optical phenomena, which is based on geometrical optics (ray optics) and makes extensive use of the paraxial approximation." Useful for calculating focal lengths.
4. What is a "gaussian blur" and how is it used in digital image editing?
According to Adobe's website, it's "the application of a mathematical function to an image in order to blur it." It's used to highlight certain parts of an image by blurring all the rest, or else to soften some or all parts of an image. Before-and-after example of softening on right.
